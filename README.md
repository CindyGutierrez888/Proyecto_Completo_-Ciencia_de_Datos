# Proyecto_Completo_-Ciencia_de_Datos
Este proyecto muestra de principio a fin el ciclo de Ciencia de Datos, desde la limpieza de datos hasta un Clustering (K-means), y un Modelo de Predicción mediante Regresion Lineal de Serie de Tiempo (SARIMAX). Ademas de incluir consultas con SQL y un visuzizador interactivo de Datos con Power BI. Las conclusiones y Documentación estan incluidas.
